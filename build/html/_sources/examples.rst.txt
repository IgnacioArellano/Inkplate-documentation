Examples
========

Here are some examples included in our Inkplate library.

Spotify API
-----------

This one is a bit harder to get working. 
Follow these steps:

    - First you need to install this library: https://github.com/witnessmenow/arduino-spotify-api
    - After you've done that follow it's installation instructions and get your refresh token, here's the summary
    - To do that you need to login in to Spotify developers
    - Create an app
    - Copy your app credentials to refresh token example and run it
    - Whitelist your ESPs ip that was written to serial monitor
    - Go to your ESPs ip and click login 
    - If it all worked well you should see the token
    - Copy that token alongside your app credentials and WiFi AP info and upload to Inkplate
